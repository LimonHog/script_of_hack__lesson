# Взлом электронного дневника
Этот скрипт содержит в себе несколько функций для взлома и внесения изменений в содержание определенного электронного дневника.

Программа работает совместно с Электронным дневником школы. Для работы потребуется база данных взламываемого дневника.


## Как он работает

Скрипт выполняет три основных действия: исправление оценок, удаление замечаний, добавление рекомендаций.

### 1. Исправление оценок

Находит все *плохие* оценки ученика (те, что ниже 4 (`points__lt=4`)), и заменяет их на пятёрки с помощью `update()` — быстро и без лишних запросов к базе.

### 2. Удаление замечаний

Эта функция удаляет все записи вида "Замечание" из дневника ученика. Это делается через фильтр по *`schoolkid`* и с помощью метода `delete()`.

### 3. Добавление похвалы

- Создаёт новую запись *`commendation`* c:
  - Случайным текстом из списка похвал.
  - Датой урока.
  - Предметом и учителем, ведущим этот урок.

## Поиск ученика:

- Происходит по его ФИО (или части ФИО, но чем больше информации, тем лучше) с поддержкой регистронезависимого поиска (*`contains`*).
- Если найдено несколько учеников — скрипт остановится и попросит уточнить имя.
- Если ученик не найден — скрипт сообщит вам об этом.

## Примеры использования

Для использования потребуется положить *файл БД дневника* в директорию со скриптом. Для запуска скриптов на windows:

- откройте командную строку
- перейдите в директорию со скриптом
- используйте команду `python manage.py shell` для запуска Django shell
- импортируйте скрипты командой `from scripts import fix_marks, remove_chastisements, create_commendation`
- запустите скрипт, введя его название и необходимые аргументы, например:
  - Команда `fix_marks("Иван Фролов")` исправит все плохие оценки у Ивана Фролова
  - Команда `remove_chastisements("Иван Фролов")` удалит все замечания, адресованные Ивану Фролову
  - Команда `create_commendation("Иван Фролов", "Музыка")` добавит похвалу Ивану Фролову за урок музыки
